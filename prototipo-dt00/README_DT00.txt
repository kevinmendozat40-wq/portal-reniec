DT00 - Prototipo listo para GitHub Pages

- DT00_CORREGIDOS.html: copia directa del prototipo.
- data/registros.csv: fuente recomendada para Power BI.

Subida correcta a GitHub:
1. Descomprimir este ZIP.
2. Subir la carpeta completa "prototipo-dt00" en la raíz del repositorio portal-reniec.
3. Verificar que GitHub muestre rutas como: prototipo-dt00/index.html y prototipo-dt00/data/registros.csv.
4. Commit sugerido: Subir prototipo DT00 listo para GitHub.
