Sincronización automática con GitHub

Use esta pantalla para que cada registro actualizado en el prototipo se escriba en prototipo-dt00/data/registros.csv y prototipo-dt00/data/registros.json.
GitHub + Power BI
Importante: GitHub Pages solo muestra la web. Para escribir datos en el repositorio se necesita un token de GitHub con permiso de escritura sobre el repositorio portal-reniec. El token no se incluye en el HTML ni se sube al repositorio; lo ingresa el usuario autorizado desde su navegador.
Pendiente de configuración.

Enlaces operativos

El CSV para Power BI quedará en: https://raw.githubusercontent.com/kevinmendozat40-wq/portal-reniec/main/prototipo-dt00/data/registros.csv

Cómo debe funcionar

1. Configure el token de GitHub una sola vez en este navegador. 2. Registre o edite atenciones en el módulo Registro. 3. Si la sincronización automática está activa, el prototipo actualiza registros.csv y registros.json en GitHub. 4. Power BI se conecta al CSV raw de GitHub y actualiza las gráficas. 5. Si trabaja desde otra PC, use “Cargar data desde GitHub” para traer la última base publicada.
